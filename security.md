# Security Policy

## Supported Versions

This project is maintained for the current v4 release line only.

## Reporting a Vulnerability

If you believe you have found a security issue, report it privately through GitHub Security Advisories or open a private issue if advisories are not available.

Please include:

- The plugin version
- Server version and Java version
- Steps to reproduce the issue
- Relevant logs, stack traces, and configuration snippets

Do not publish exploit details publicly until the issue has been reviewed and a fix is available.
