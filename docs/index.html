Overview

ResourceWorldResetter is a powerful Spigot plugin that automates resource world resets with phase-aware status tracking, an intuitive admin GUI, and a unified command structure.

MIT License Java 21+ Spigot API 1.26 v4.0.0

Installation

Requirements

  • Java 21 or higher
  • Spigot/Paper/Purpur server (API version 1.26+)
  • Multiverse-Core (recommended, but optional)

Setup Steps

  1. Stop your server
  2. Back up your worlds and plugin folder
  3. Place the plugin JAR in the plugins/ folder
  4. Start your server
  5. Configure using /rwr gui or edit config.yml
  6. Verify setup with /rwr status

Key Features

  • Automated Resets - Daily, weekly, or monthly schedules
  • Region Resets - Reset specific areas without affecting the entire world
  • Admin GUI - Intuitive in-game configuration interface
  • Phase Tracking - Explicit reset phases for better visibility
  • Safe Teleportation - Players moved to safe locations before resets
  • Failure Recovery - Auto-resume capabilities for interrupted resets
  • Multiverse Integration - Seamless world management
